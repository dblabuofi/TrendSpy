Current version support Three different types of queries.

Key Words Description
Event Type Point event: maximum 120 days
Range event: maximum 365 days
Key words Phrases are separated by comma. For example: Mt Rainier, Paradise Ice Caves returns any searches include exact phrase Mt Rainier, or Paradise Ice Caves
Primary Location supported US location only
Observation Start Date Time to start recording
Observation End Date Time to End recording
Event Start Date Event start
Event End Date Event end
